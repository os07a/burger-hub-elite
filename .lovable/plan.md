# خطة: توليد المشروع كملف HTML تفاعلي واحد

## الهدف
دمج الوثائق الثلاث (الخريطة الذهنية + الهيكلة + الثغرات) في **ملف HTML واحد** أنيق، RTL، يفتح بأي متصفح بدون إنترنت تقريباً، وقابل للطباعة كـ PDF.

## المخرج
ملف واحد في `/mnt/documents/`:
- **`burgerhum_project.html`** — صفحة واحدة شاملة

## محتوى الصفحة

### 1. Header (هوية برجرهم)
- شعار نصي + ألوان الهوية (Crimson #B03030، Charcoal #2C2C2C، Gold #C4A265)
- خط SF Pro / Tajawal للعربية
- تاريخ التوليد + ملخص سريع (17 صفحة، 28 hook، 28 جدول، 10 Edge Functions)

### 2. شريط تنقل علوي ثابت (Sticky Tabs)
4 تبويبات بنقرة واحدة:
- نظرة عامة | الخريطة الذهنية | الهيكلة التفصيلية | الثغرات والتحسينات

### 3. قسم نظرة عامة
- 6 بطاقات KPI (الصفحات، Hooks، الجداول، Edge Functions، التكاملات، قواعد الأعمال)
- ملخص نظام الـ 200 سهم
- حالة المشروع الحالية

### 4. قسم الخريطة الذهنية
- عرض الـ Mermaid mindmap مباشرة عبر `mermaid.js` (CDN واحد)
- إمكانية التكبير/التصغير
- نسخة شجرية HTML احتياطية لو ما اشتغل Mermaid

### 5. قسم الهيكلة التفصيلية
- جدول: الصفحة → Hooks → الجداول → Edge Functions
- مصفوفة التكاملات الخارجية
- خريطة تدفق البيانات (ASCII داخل `<pre>`)

### 6. قسم الثغرات والتحسينات
- 3 ألوان: أحمر (حرج)، أصفر (متوسط)، أخضر (تحسين)
- بطاقات قابلة للطي (`<details>`)
- أولويات مرتبة

### 7. Footer
- زر طباعة كـ PDF
- إصدار الوثيقة

## التقنية
- **HTML5 + CSS مدمج** (بدون ملفات خارجية عدا Mermaid CDN)
- **RTL كامل** مع `dir="rtl"` و Tajawal font من Google Fonts
- **Responsive** يعمل على موبايل وديسكتوب
- **Self-contained**: كل الـ CSS و JS داخل الملف
- **Print-friendly**: `@media print` لطباعة نظيفة كـ PDF

## ما لن أفعله
- لن أعدّل أي كود في المشروع
- لن أغيّر قاعدة البيانات
- مجرد توليد ملف HTML من البيانات اللي جمعتها سابقاً

## بعد الموافقة
أحوّل لـ default mode، أبني الملف وأعرضه كـ artifact قابل للتنزيل والمعاينة مباشرة في المتصفح.
